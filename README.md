### Tech Stack

- **Electron 28.0** - Cross-platform desktop framework

### Build Commands

```bash
# Development with hot-reload
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Package as executable
npm run package

# Run production build
npm start
```
